### TC8

* 데이터 파일이 보조기억장치에 저장되는 방법이나 저장된 데이터의 접근 방법을 변경할 때는 응용 프로그램도 같이 변경해야 한다.
* 데이터 파일의 저장 방법이나 접근 방법이 바뀔 때는 프로그램도 같이 바꿔야 한다.
* 데이터파일을 기억장치에 저장하는 법이나 저장한 데이터에 접근하는 법이 바뀔 때는 응용프로그램도 같이 바뀌어야 한다.
* 데이터를 보조기억장치에 저장하는 법, 데이터에 접근하는 방법을 변경하면 응용 프로그램도 바꿔야 한다.
* 보조기억장치에 데이터를 저장하는 법, 혹은 데이터를 접근하는 방법을 변경하면 그걸 쓰는 프로그램도 같이 변경해야 함
* 보조기억장치에 파일을 저장하는 방법 또는 그 파일에 접근하는 방법을 바꾸면 응용어플리케이션도 같이 바꿔야 한다.
* 파일 접근 방법 변경, 저장방법 변경 -> 응용 프로그램도 바꿔줄 것
